# 权限修改功能操作手册

## 功能简介

在远程 SSH 文件浏览器中，可以直观地修改文件或目录的读/写/执行权限。

## 使用方法

1. 在 SSH 资源管理器中找到目标文件或目录
2. **右键点击**文件或目录，选择 **SSH: 修改权限**
3. 弹出权限编辑面板：
   - 表格分为三行：**所有者**、**用户组**、**其他**
   - 每行有三个复选框：**读**、**写**、**执行**
   - 点击复选框即可切换对应权限
4. 点击 **Apply** 应用修改，或点击 **Cancel** 取消操作
5. 修改成功后会弹出提示：`Permission set to 755`（示例）

## 八进制预览

编辑面板顶部会实时显示当前选择的八进制权限值（如 `0755`、`0644`），方便确认。

## 权限说明

| 权限位 | 含义 | 说明 |
|--------|------|------|
| 读 (Read) | 查看文件内容 / 列出目录内容 | |
| 写 (Write) | 修改文件内容 / 创建删除目录内文件 | |
| 执行 (Execute) | 运行文件 / 进入目录 | 目录需要执行权限才能 cd 进入 |

| 角色 | 说明 |
|------|------|
| 所有者 (Owner) | 文件的所有者用户 |
| 用户组 (Group) | 文件所属用户组 |
| 其他 (Other) | 既不是所有者也不在用户组的其他用户 |

## 常见问题

- **修改权限失败**：确认 SSH 连接已建立，且你有足够的权限（通常需要文件所有者身份）
- **点击 Apply 无反应**：检查网络连接是否正常，可尝试重新连接后重试
- **权限显示未更新**：右键刷新或重新展开目录即可
